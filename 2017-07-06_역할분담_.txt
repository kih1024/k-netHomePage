동적 웹 페이지 제작 위한 bootstrap 사용

Web 양식 :
Title 의 제목 클릭 할경우 메인 화면으로 이동하도록 구현.
Title, 메뉴 format은 변하지 않도록 고정.

메인 화면의 경우 공지사항, 과제 Q/A게시판, 자유 게시판, 활동 사진[4개정도 게시글 업로드]
└> 교체 될 수 있는 부분

Title 상단에 Login, 회원가입 버튼 삽입 <- 타 폼으로 이동하더라도 변하지 않도록

+ 세미나의 경우 팀 비밀번호를 게시글에 설정하여 해당 팀원만 열 수 있도록 변경

+ 자료실의 경우 정회원의 등급 이상의 사람들 접근 가능하도록 설정.

+ 메인 화면에 이미지가 포함된 게시글의 경우 아이콘을 이용하여 사진포함 여부를 표시.

+ 자료실의 접근권한의 경우 추후 Test를 통해 설정할 예정

+ git에 Pull을 하는데에 있어 미리 카톡통해 알리고 Pull 할 것.

+ 페이지에 대한 Layout이 나온다면 그에 따른 코드 작성.


+ 추가적인 필요한 사항 : 각 구성별 필요 정보, 기능 에 대한 DB 관계도 작성

+ 게시판 Flow Chart 통해 : 각 게시판별 이동 사항, 필요 파일 생성.
