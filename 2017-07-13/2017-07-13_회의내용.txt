각 기능 Layout

===메인 화면===
+ 전체 보기 버튼을 만들어 상단 메뉴가 밑의 화면을 밀면서 Sub메뉴를 보여줌
+ 상단메뉴를 스크롤바에 따라 고정하여 움직이도록 구현
+ 단체 사진(동적)+[이벤트 없음] 및 공지사항, 과제 Q/A 게시판, 자유 게시판 올리기 + 갤러리
└> '+' 버튼을 통해 해당 게시판으로 이동 가능
<왼쪽 화면 위치 기능>
+ 로그인
+ 이달의 생일자 날짜 별로 인원 보여주기
+ 통합검색 (돋보기 아이콘을 두기) => 좌측 고정 창에 위치
└> 검색 기준 (제목, 내용, 작성자) -> 내용 검색 방법에 대해 자문해볼 것
+ 페이스북 창을 불러와 일정 Layout내에 표시
참고 : http://www.onad.me/

===게시판 화면===
+ 글번호, 태그, 제목, 조회수, 작성일시, 작성자 -> Table로 표시하는 것이 좋을듯.
└> 사진, 익명 게시판 제외 게시판

===공지사항===
+ 글쓰기 버튼이 관리자에게만 보이도록 할 수 있는지 의문
+ 공지사항 중 상단 고정 공지 추가 하면 좋을 듯
+ 글쓰기 폼은 알아볼것.
+ 파일 업로드 기능 이용하여 파일 추가 가능

===세미나 자료실===
+ 글 삭제, 수정 의 경우 관리자, 작성자에 한해 보이도록 할 수 있는지 의문
+ 태그를 (세미나 종류) 와 같이 표기하여 검색에 용이하도록 설정
+ U-Campus 의 Layout을 차용하여 작성
└> 세미나 선생님들의 경우 권한을 주어 해당 세미나만 관리가 가능하도록
+ 세미나 인원들의 경우 자신이 참여하고 있는 세미나에 대해 선택이 가능하도록
+ 해당 세미나의 강의자료는 일반 게시판과 같이 올라감 

===프로젝트 자료실===
+ 일반 게시판과 같이 나열
└> 최종결과를 올리는 식으로 게시판을 작성자를 작성

===Q&A 자료실===
+ 태그는 언어 종류 사용,
+ 해당 게시글에 조언, 댓글의 경우는 Google Docs, pdf에 메모하는 것처럼 구현하면 좋을듯.

===갤러리===
+ 제목, 댓글, 업로드 날짜 포함된 대표사진 한장 표시 -> 2열로 View
└> 관리자가 업로드
+ 기타 게시판 게시글 작성 방식 가져오기
+ 댓글 작성가능

===게시판===
+ 게시글 내부에 동영상 삽입시 Youtube영상 바로 재생가능하도록
+ 자유게시판의 경우 익명 체크 가능하도록 
+ 익명게시판의 경우 에브리타임 Layout을 차용하여 간단하게 작성
